Homework
--------

On Wednesday, we will start to program. Please make sure you have the
following prerequisites installed:

* In WSL, install the packages ``build-essential`` (C and C++
  compilers) and ``cmake`` (a build tool at a higher level than
  ``make``)
* In Windows, install `Visual Studio Code
  <https://code.visualstudio.com/docs/cpp/config-wsl>`__
