Exercise: Even Numbers (``while``)
==================================

#. Write a program that outputs all even numbers from 0 (inclusive) up
   to a user-supplied upper bound (exclusive).

   Use a ``while`` loop!

.. code-block:: console

   $ python even.py 10
   2
   4
   6
   8
