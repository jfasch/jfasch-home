Exercise: Dictionary: Errors?
=============================

In the interactive interpreter, create a dictionary like so:

.. code-block:: console

   >>> d = {
   ...     'one': 1,
   ...     'two': 2,
   ... }
   >>> d['one']
   1

What happens if you access a non-existent key?

.. code-block:: console

   >>> d['three']
