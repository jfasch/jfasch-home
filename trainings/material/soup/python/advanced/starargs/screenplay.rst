And Keyword Arguments?
----------------------

* ``*args`` |longrightarrow| no parameter names
* no keyword arguments

   :language: python

* A-ha: similar to ``*args``, ``**kwargs`` is passed as dictionary
* And unpacking |longrightarrow| same: ``**``
